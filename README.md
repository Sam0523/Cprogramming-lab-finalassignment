### PART 1

- These characters are considered to delimit words:
```
 ,;:()[]{}<>\"\n\t@#$%^&*_+=
```
**Note**: hyphen are **NOT** considered to be same as a space!

- I used binary search tree instead of linked lists so as to improve
performance.
